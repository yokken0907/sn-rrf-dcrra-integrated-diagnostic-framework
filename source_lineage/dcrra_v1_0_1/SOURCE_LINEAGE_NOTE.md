# Source lineage note: DCRRA v1.0.1

This folder preserves the prior DCRRA v1.0.1 repository package as a parent lineage source for the v3.0.1 complete integrated repository.

Role in v3.0.1:
- establishes the seven-zone residual-regime map for 143 eligible SPARC galaxies;
- preserves the DCRRA claim boundary that the map is a diagnostic taxonomy, not a physical dark-matter taxonomy;
- supplies manuscript, figures, outputs, scripts, and claim-boundary materials used as source lineage.

This folder is not the active root package for the v3.0.1 release. Active v3.0.1 files are at the repository root and in `paper/`, `outputs/`, `docs/`, `scripts/`, and `tools/`.
