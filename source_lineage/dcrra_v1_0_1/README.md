# Claim-Bounded Diagnostics for Dark-Component-like SPARC Rotation-Curve Residuals

This repository contains the **Dark-Component Residual Response Audit (DCRRA)** package: a claim-bounded diagnostic audit of dark-component-like residual structure in SPARC galaxy rotation-curve data.

## Core result

DCRRA0-DCRRA8 support a diagnostic residual-regime map, **not** a physical dark-matter taxonomy.

- eligible SPARC galaxy count: 143
- residual-map zone count: 7
- response-family-associated count: 77
- halo-comparator-dominant count: 6
- mixed three-way competitive count: 16
- pathology or nuisance pass-through count: 42
- unresolved triangular count: 2
- DCRRA8 guard status: 9/9 passed

## Claim boundary

This project does **not** claim dark-matter exclusion, Lambda-CDM replacement, MOND/RAR falsification, NFW-halo falsification, Bullet-Cluster explanation, particle identity, Hubble-tension solution, or SN-RRF as a physical dark-matter substitute.

The allowed interpretation is narrower: under the reported DCRRA protocol, dark-component-like SPARC rotation-curve residuals can be organized into multiple diagnostic residual regimes rather than treated as one homogeneous category.
