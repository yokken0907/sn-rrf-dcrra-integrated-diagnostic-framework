# 暗黒成分的SPARC銀河回転曲線残差の主張境界付き診断

本リポジトリは、**Dark-Component Residual Response Audit (DCRRA)** の公開パッケージです。SPARC銀河回転曲線における暗黒成分的な残差構造を、主張境界付きで診断分類します。

## 中核結果

DCRRA0-DCRRA8は、物理的な暗黒物質分類ではなく、診断的な残差レジーム地図を支持します。

- 対象SPARC銀河数: 143
- 残差地図のゾーン数: 7
- response-family-associated count: 77
- halo-comparator-dominant count: 6
- mixed three-way competitive count: 16
- pathology or nuisance pass-through count: 42
- unresolved triangular count: 2
- DCRRA8 guard status: 9/9 passed

## 主張境界

本研究は、暗黒物質否定、Lambda-CDM置換、MOND/RAR反証、NFWハロー反証、弾丸銀河団説明、粒子正体主張、ハッブルテンション解決、SN-RRFの物理的暗黒物質代替化を主張しません。

許容される解釈はより限定的です。すなわち、報告されたDCRRAプロトコルの範囲内で、暗黒成分的SPARC回転曲線残差を、単一の均質な分類ではなく、複数の診断的残差レジームとして整理できる、というものです。
