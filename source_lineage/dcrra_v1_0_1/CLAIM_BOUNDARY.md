# Claim Boundary

This repository is a claim-bounded diagnostic package for SPARC galaxy rotation-curve residual-regime mapping.

## Allowed claim

The reported DCRRA protocol organizes dark-component-like SPARC rotation-curve residuals into multiple diagnostic residual regimes under the tested comparator, proxy-audit, and null-control workflow.

## Not claimed

This repository does not claim:

- dark-matter exclusion;
- Lambda-CDM replacement;
- MOND/RAR falsification;
- NFW-halo falsification;
- Bullet-Cluster explanation;
- dark-matter particle identity;
- Hubble-tension solution;
- SN-RRF as a physical substitute for dark matter;
- independent observational validation outside the tested SPARC diagnostic setting.

## DOI interpretation

Zenodo identifiers generated for this project should be interpreted as repository-archive DOIs for versioned GitHub releases. They are not standalone peer-reviewed article DOIs and do not broaden the claim boundary.
