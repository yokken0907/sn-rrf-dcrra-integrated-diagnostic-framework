# Claim-Bounded Diagnostics for SPARC Galaxy Rotation-Curve Residuals

**Repository short name:** `SN-RRF / A10-TVC-RRF diagnostic audit`  
**Version:** `v2.0.1-claim-bounded-reader-guidance`  
**GitHub URL:** https://github.com/yokken0907/a10-tvc-rrf-galaxy-residual-diagnostic  
**Prior repository-archive DOI:** https://doi.org/10.5281/zenodo.20224479

This repository contains the integrated manuscript and supporting materials for a claim-bounded diagnostic audit of SPARC galaxy rotation-curve residuals.

The project begins with an A10/TVC-inspired fixed-kernel residual-response diagnostic and then generalizes the surviving signal into **SN-RRF: Scale-Normalized Residual-Response Family**.

## Reader guidance

This package should be read as a diagnostic and reproducibility-oriented repository, not as a final astrophysical or cosmological theory.

The Zenodo DOI associated with any GitHub/Zenodo release of this repository identifies an archived repository release. It should not be interpreted as a standalone peer-reviewed article DOI.

## Main conclusion

The cautious integrated conclusion is not fixed-exponent uniqueness and not a dark-matter replacement claim.

The supported diagnostic interpretation is:

> Within the tested SPARC residual-audit protocol, galaxy rotation-curve residuals exhibit evidence consistent with a scale-normalized monotone-saturating residual-response family. In dimensionless response space, non-pathology classes concentrate around a central manifold, while core-pathology cases are separated by a pathology moat.

## Key numerical checkpoints

- Valid SPARC eligible population: 143 galaxies.
- A10/TVC-RRF population relation: `rrf_log10_A` vs `v_max_kms`, Spearman rho = 0.873, permutation p = 0.001996.
- RAR-relative diagnostic audit: median Delta AIC(TVC-RRF - RAR-like) = -33.482; beats-RAR fraction = 0.867.
- Stage 23 manifold stability: 7/7 guards passed.
- Stage 26 curve-level destructive null controls: 5/5 guards passed; RRF beats best destructive null fraction = 0.916.
- Stage 30 kernel-continuum profile: 6/6 guards passed; best-n median = 2.25; best-n IQR = [1.5, 3.2545].

## Claim boundary

This repository does **not** claim:

- dark-matter exclusion;
- Lambda-CDM replacement;
- MOND/RAR falsification or defeat;
- Bullet-Cluster explanation;
- proof of a physical TVC mechanism;
- proof that `n = 3.258993` is a universal physical exponent;
- a Hubble-tension solution;
- independent observational validation outside the supplied audit package.
