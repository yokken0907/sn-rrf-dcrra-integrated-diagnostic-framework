# Stage 30 Optional Kernel-Continuum Profile

Stage 29 is sufficient to repair the theory language from fixed-exponent SN-RRF to family-level SN-RRF.

A possible Stage 30 extension would refit a continuous shape parameter rather than using the discrete kernel set in Stage 28. This would test whether the preferred saturating-kernel shape clusters near a stable exponent range such as `n ~ 2--4`, or whether it is broadly degenerate.

This stage is optional and should not be used to claim physical uniqueness without much stronger external validation.
