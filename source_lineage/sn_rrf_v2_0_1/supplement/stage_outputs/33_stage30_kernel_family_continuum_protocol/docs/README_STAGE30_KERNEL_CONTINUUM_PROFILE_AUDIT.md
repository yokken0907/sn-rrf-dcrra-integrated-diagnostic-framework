# Stage 30 Kernel-Family Continuum Profile Audit

## Purpose

Stage 28 showed that the locked fixed-exponent kernel is not unique among monotone
saturating neighbors. Stage 29 therefore repaired the theory language from a fixed exponent
model into a scale-normalized residual-response family.

Stage 30 tests this family-level interpretation more directly by scanning a continuous grid of
saturating exponents:

```text
K_n(x) = x^n / (1 + x^n)
```

with `A` and `r_c` refit for each galaxy and each exponent.

## What this stage can support

Stage 30 can support:

```text
SN-RRF is better treated as a heterogeneous saturating residual-response family than as a
single universal fixed-exponent law.
```

## What this stage cannot support

Stage 30 cannot support:

- dark-matter exclusion,
- Lambda-CDM replacement,
- MOND/RAR defeat,
- Bullet-Cluster explanation,
- proof of a physical TVC mechanism,
- solution of the Hubble tension,
- a universal fixed exponent.
