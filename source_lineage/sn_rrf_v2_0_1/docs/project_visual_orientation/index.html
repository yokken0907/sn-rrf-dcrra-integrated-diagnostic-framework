0. Overview

The old TVC-scale interpretation is not retained. The surviving object is a residual-response diagnostic that generalizes into SN-RRF.

Safe conclusion:
SPARC galaxy rotation-curve residuals support a scale-normalized monotone-saturating residual-response family diagnostic under the tested protocol.
Not claimed:
No dark-matter exclusion, Lambda-CDM replacement, MOND/RAR defeat, physical TVC proof, universal fixed exponent, or Hubble-tension solution.
