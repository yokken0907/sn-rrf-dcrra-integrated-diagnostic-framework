## v2.0.1 - Claim-Bounded Reader Guidance and Repository DOI Clarification

This release revises the SN-RRF / A10-TVC-RRF galaxy rotation-curve residual diagnostic repository to improve reader guidance, claim-boundary clarity, and repository-archive DOI handling.

### Main changes

- Reframed the manuscript title toward claim-bounded diagnostics and null-control audits.
- Added stronger reader guidance before the technical construction.
- Clarified that Zenodo identifiers generated for this repository identify archived repository releases, not standalone peer-reviewed article DOIs.
- Preserved the prior v2.0.0 DOI/history for traceability.
- Updated README, README_ja, CLAIM_BOUNDARY, CITATION.cff, and `.zenodo.json` for the new release.
- Archived the prior v2.0.0 manuscript framing under `paper/archive_v2_0_0_original_framing/`.
- Regenerated the repository manifest and verified package consistency.

### Scope

This release does not claim dark-matter exclusion, Lambda-CDM replacement, MOND/RAR falsification, Bullet-Cluster explanation, proof of a physical TVC mechanism, universal fixed-exponent law, independent observational validation, or a Hubble-tension solution.

### DOI note

Do not enter an existing `10.5281/zenodo.xxxxx` value as an external DOI when creating the Zenodo release. Select DOI generation by Zenodo. The resulting DOI should be treated as a repository-archive DOI.
